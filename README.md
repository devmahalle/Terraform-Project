# Terraform AWS Web App

## 🚀 Project Overview
This project provisions a complete AWS infrastructure using Terraform.

## 🧩 Features
- Custom VPC
- Public Subnet
- Internet Gateway
- Route Table
- Security Groups
- EC2 with Apache Web Server

## 🛠️ Tech Stack
- AWS
- Terraform

## ▶️ How to Run
terraform init
terraform apply

## 🌐 Output
Public IP of deployed web server
